# CV Maker API

## Project setup

Set these ENV variables
- CLOUDINARY_KEY
- CLOUDINARY_NAME
- CLOUDINARY_SECRET
- DB_SERVER
- DB_NAME
- DB_USER
- DB_PASSWORD
- SECRET
```
npm install
npm start
```

### Compiles for production

```
npm build
```
